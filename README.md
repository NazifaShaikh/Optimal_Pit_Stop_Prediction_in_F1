# Optimal Pit Stop Prediction in F1



This project uses a GRU-based deep learning model to predict the number of laps until the next pit stop in Formula 1 races using lap-by-lap data from the 2024 season.



### Project Structure



project/ │── src/ │── docs/ │── requirements.txt │── README.md





### How to Run



pip install -r requirements.txt python src/data\_preprocessing.py python src/models\_gru.py
